rgxll *&  root 
)

Regex pattern lexical analyzer.

Parameters
inputA list of scan symbols to parse into a node tree.
rootThe resultant root node, returned by reference, or NULL if invalid regex syntax. Note that root is dynamically allocated, so it must be deleted after calling if not NULL.
Returns
A flag representing the first error encountered. If no errors, returns RGX_NO_ERROR, which is equivalent to 0.
