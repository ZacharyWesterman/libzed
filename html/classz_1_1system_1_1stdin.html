Detailed Description

Class for getting input from stdin.

This class provides an implementation of core::inputStream. All read-related methods read from stdin. Note that because of how stdin behaves, you cannot unget characters or seek a position in the stream; those are just dummy methods. As a result, this class is not compatible with the util::regex class.
