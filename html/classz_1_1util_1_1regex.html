Detailed Description

template<encoding E>
class z::util::regex< E >

A class for parsing and matching regular expressions.

Note that because the class requires seeking locations in a stream, it will not work with the system::console class.

RE-ENTRANCE: Multiple accesses to the same object can cause data races.
