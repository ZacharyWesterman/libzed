Detailed Description

Regex rule interface.

This class contains default data that any regex rule must contain, and sets default rule behavior.

◆ greedy

bool z::util::rgx::rule::greedy

Whether this rule should consume input greedily.

If false, will match until a sibling rule meets its minimum. If true, this rule matches as many times as it can.
