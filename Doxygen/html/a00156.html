Detailed Description

template<typename CHAR>
class z::core::string< CHAR >

A template for character strings.

Strings are expected to be of either char or wchar_t types, but other integer types should work. This class is type-independent, so it handles all type conversions automatically.

Constructor from complex numeric type.

If the parameter is of type std::complex, then it is converted to a string with the format a+bi. Note that if either the real or imaginary part are zero, then only the non-zero part is converted. If both parts are zero, then the resultant string is "0".
